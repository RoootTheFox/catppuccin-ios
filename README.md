## Usage

1. Download the .deb from the releases tab
2. Install it using your package manager (Sileo, Zebra, etc)
3. Respring, go to settings and select your prefered flavor/accent

<!-- this section is optional -->
## 🙋 FAQ

-	Q: **_"How do i change the accent color and/or flavor?"_**\
	A: In the Settings app, there should be a section labeled "Catppuccin". In there you can change both flavor and accent color.
